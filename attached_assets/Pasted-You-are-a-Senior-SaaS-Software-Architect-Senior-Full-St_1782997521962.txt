PROJECT CONTEXT
====================================================

This is NOT a new project.

This is an existing, fully working ERP system that was originally developed for a single HVAC/Air Conditioning company named "JCAJ Cooling Solutions".

The objective is to transform this existing custom ERP into a professional, production-ready, Multi-Tenant SaaS ERP Platform that can be used by multiple Air Conditioning, HVAC, Refrigeration, and Mechanical companies using a single codebase.
